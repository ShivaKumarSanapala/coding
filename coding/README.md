## How to Run

1. Make sure you're in the project root:

   ```bash
   cd path/to/coding
   ```

2. Compile the project using Maven:

   ```bash
   mvn clean compile
   ```

3. Run the main class:

   ```bash
   mvn exec:java -Dexec.mainClass="com.personal.Main"
   ```
